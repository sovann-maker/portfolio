### Prerequisites

- Flutter SDK (3.0+)
- Dart SDK

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
flutter pub get

# Run development server
flutter run -d chrome --web-port 3000

# Build for production
flutter build web
```
